    HTML

    Curso "Introducción al desarrollo web": HTML (HyperText Markup Language) se emplea únicamente para definir la estructura y el contenido o información de una página web, la presentación visual no se define con HTML, estructura de un elemento (etiqueta), ¿cómo se aprende HTML?, etiquetas básicas (html, head, title, body, h1, h2, h3, h4, h5, h6, p, strong, em).

    Obtenido de https://www.youtube.com/watch?v=Ch1GG4xM8nw/

    HTML es el lenguaje de marcado estándar para crear páginas web.

    HTML significa lenguaje de marcado de hipertexto

    HTML describe la estructura de una página web

    HTML consiste en una serie de elementos

    Los elementos HTML le dicen al navegador cómo mostrar el contenido

    Los elementos HTML están representados por etiquetas

    Las etiquetas HTML etiquetan piezas de contenido como "encabezado", "párrafo", "tabla", etc.

    Los navegadores no muestran las etiquetas HTML, pero las usan para representar el contenido de la página

    CSS

    9 Tips o buenas prácticas o reglas con el CSS.

    Primero se definen los estilos para etiquetas. Ejemplo: header,section,nav,div,a,p,etc.

    Después se definen estilos para Identificadores(id).Los ID son elementos unicos en Html, elementos que solo se usan una vez en el Codigo html, en el css se declara con el simbolo # ejemplo: #nombre-id{ estilos } y en el html se llaman mediante el atributo id, ejemplo

    .

    Al final se definen los estilos para las clases (class), las clases son elementos que se pueden repetir 'N' veces en el html, en el css se declara con el simbolo de '.' ejemplo: .nombre-clase{ estilos } y en el html se llaman mediante el atributo class, ejemplo

    .

    Todos los estilos se ordenan conforme van apareciendo en la disposición del html. Ejemplo de index.html anterior, body, header, div id, div class, footer. Teniendo en cuenta prioridad anterior, es decir, primero etiquetas html, luedo id's y al final clases.

    Los atributos se declaran de la siguiente forma: {atributo1: valor1; atributo 2:valor2; ...; atributoN:valorN;}

    Todos los atributos se ordenan alfabéticamente, por ejemplo si tengo: width, color, display, entonces el orden sería: color, display, width.

    Si tenemos estilos para las etiquetas html y/o body estos estilos son los primeros en definirse.

    En el body debemos declarar el tamaño de letra global.

    Si tenemos estilos para resetear elementos html, estos irán antes de cualquier estilo.

    El asterisco significa todos los elementos, es un comodín y aplica todos sus estilos a todos los elementos.
